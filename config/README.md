# Configuration Directory

This directory contains runtime configuration files and calibration data.

## Files

### calibration.json
Sensor calibration data including:
- RPM pulses per revolution
- Tank level resistance curves
- Temperature sensor curves
- Pressure sensor curves

### sensors.json
Sensor configuration:
- Enabled sensors
- Pin assignments
- Sensor names and units

### wifi.json
WiFi network credentials (encrypted)

### victron.json
Victron device connections:
- Paired BLE devices
- VE.Direct serial ports
- Device names and types

## Note

These files are automatically generated by the application and should not be edited manually. Use the Settings page in the web interface to configure these values.

Configuration files are excluded from git for security reasons (.gitignore).
